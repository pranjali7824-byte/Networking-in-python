# Python TCP Client-Server Chat Application

A simple networking project built using Python's socket module that enables communication between a client and a server over TCP.

## Features
- Client-Server communication
- TCP socket programming
- Real-time message exchange
- Exit command to close the connection

## Technologies Used
- Python
- Socket Programming
- TCP/IP Protocol

## Files
- `s.py` - Server Program
- `c.py` - Client Program

## How to Run

### Start Server
```bash
python s.py
