## 🚀 About The Project

This project is a GUI-based Client-Server Chat Application that enables real-time communication between a client and a server over a network. 

The application uses Python sockets for communication and Tkinter for creating an interactive graphical user interface.

---

## ✨ Features

- ✅ **Real-Time Messaging** - Instant message delivery between server and client.
- ✅ **Client-Server Architecture** - Structured network communication model.
- ✅ **Tkinter GUI Interface** - User-friendly window layout instead of a boring terminal.
- ✅ **Multi-Threaded** - Handles sending and receiving simultaneously without freezing the GUI.
- ✅ **Fast & Lightweight** - Consumes minimal system resources.
